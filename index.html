Macachor Sovereign OS (MSOS)

Deterministic governance kernel with provably orthogonal, machine‑enforceable rights.

Overview

  • Enforces rights as logical predicates.
  • Guarantees orthogonal permission composition.
  • Records every action in an immutable Merkle‑chained audit log.
  • Runs deterministically, eliminating race‑condition exploits.

Orthogonality Ontology

The ontology is published as an OWL file at https://macachor.org/ontology/orthogonality.

Key Features

  • Deterministic Governance Kernel – mathematically provable state transitions.
  • Machine‑Enforceable Rights – predicates automatically reject illegal actions.
  • Orthogonal Permission Model – composable without hidden side‑effects.
  • Formal Protocol Verification – protobuf schemas verified by SMT solvers.
  • Immutable Audit Log – Merkle‑tree hash‑chaining for tamper evidence.

Live Demo

Open the developer console and try:

// Register a right for Alice
msos.registerRight("alice", "can_read(resource_A)");

// Attempt an illegal write
msos.request("alice", "write(resource_A)");
// → {code:403, detail:"can_read(resource_A) violated"}
